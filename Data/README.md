## Description

The dataset includes:
- Hourly WBGT-related features (temperature, humidity, wind speed, etc.)
- Target variable: WBGT (Wet Bulb Globe Temperature)
- Region: 8 Divisions in Bangladesh
- Period: Jan 2013 to Sept 2022
